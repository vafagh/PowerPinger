## 🎯 چی، کەی، و بۆ کێ

### 🤔 **ئەم ئامرازە چی دەکات؟**
PowerPinger یارمەتیت دەدات **سێرڤەر و خزمەتگوزارییە دەستڕاگەیشتووەکانی ئینتەرنێت بدۆزیتەوە** کاتێک دەستڕاگەیشتنی ئاسایی ئینتەرنێتەکەت سنوورداری هەیە یان بلۆک کراوە. ئەو IP ەکان سکان دەکات بۆ دۆزینەوەی ئەوانەی هێشتا دەگەیشترێن و چ خزمەتگوزارییەک بەردەستە.

**چیت بەدەست دێت:**
- 📋 **لیستی IP ە دەستڕاگەیشتووەکان**: دۆزینەوەی سێرڤەرەکان کە پەیوەندیت پێوە دەکرێت
- 🔍 **دۆزینەوەی خزمەتگوزاری**: بزانە کەدووک پۆرت/خزمەتگوزاری لەسەر هەر IP ەک کار دەکات
- 🛡️ **شیکردنەوەی تۆڕ**: تێبگە لەوەی چ سێرڤەرێک دەگەیشترێت
- 📊 **راپۆرتی تەفصیلی**: فایلی CSV بەدەست بهێنە لەگەڵ هەموو زانیاری پەیوەندیکردنەکان

### ⏰ **کەی ئەمە بەکاربهێنیت؟**
تەواو گونجاوە بۆ ئەو بارودۆخانەی کە دەستڕاگەیشتنی ئینتەرنێتەکەت بەتوندی سنوورداری هەیە:

- 🚫 **دەستڕاگەیشتن بۆ ئینتەرنێت نییە**: تۆڕەکەت زۆربەی ماڵپەڕ و خزمەتگوزارییەکان بلۆک دەکات
- 🔒 **VPN/Proxy بلۆک کراوە**: ناتوانیت ئامرازەکانی ئاسایی تێپەڕاندن بەکاربهێنیت
- 🛡️ **لە پشت دیواری ئاگرەوە**: پێویستیت بە دۆزینەوەی ئەوەی چ پەیوەندییەک هێشتا کار دەکات
- 📡 **پەیوەندی فریاکەوتن**: پێویستیت بە ڕێڕەوی جێگرەوە بۆ خزمەتگوزارییەکانی ئینتەرنێت
- 🔧 **چارەسەرکردنی کێشەی تۆڕ**: تێگەیشتن لە کێشەکانی پەیوەندیکردن

### 👥 **ئەمە بۆ کێیە؟**
- 🔬 **لێکۆڵەران**: خوێندنەوەی سنووردارکردنەکانی ئینتەرنێت و دەستڕاگەیشتن
- 🌍 **نەماینەگانی دەستڕاگەیشتن**: بەڵگەنامەکردنی سنووردارکردنەکانی تۆڕ
- 🛡️ **پیشەیییەکانی تۆڕ**: شیکردنەوەی پەیوەندی و سنووردارکردنەکان
- � **بەڕێوەبەرانی سیستەم**: چارەسەرکردنی کێشەکانی دەستڕاگەیشتنی تۆڕ
- 👨‍💻 **هەرکەسێک لە تۆڕی سنووردار**: دۆزینەوەی پەیوەندییەکانی کارکەری ئینتەرنێت

### 📦 **بۆچی ئەم ئامرازە؟**
**PowerPinger باشترین سکەنەری تۆڕ لە بازاڕدا نییە** - بەڵام سوودەکانی تایبەتی هەیە:

- 🚀 **دامەزراندنی سفر**: تەنها ئەوە بەکاردەهێنێت کە لەگەڵ Windows 10/11 دێت
- 📧 **زۆر گواستنەوە**: ئامراز + لیستی IP = کەمتر لە 18KB کاتێک زیپ دەکرێت (دەتوانرێت ئیمەیڵ بکرێت!)
- 🔓 **کار دەکات کاتێک ئەوانی تر ناکەن**: پێویستی بە مافی بەڕێوەبەر یان نەرمەکالای تایبەت نییە
- 🎯 **بۆ مەبەست دروستکراوە**: بە تایبەتی بۆ ژینگەی تۆڕی سنووردار دیزاین کراوە
- ⚡ **دەستبەجێ ئامادە**: دەری بهێنە و لەسەر هەر کۆمپیوتەری Windows دەستبەجێ ڕایبکە

### ⚠️ **ئاگاداری گرنگ**

**🚨 مەترسییەکانی ئاسایش:**
- **دەتوانرێت چاودێری بکرێیت**: بەڕێوەبەرانی تۆڕ دەتوانن چالاکی سکانکردنەکەت ببینن
- **گونجانی یاسایی**: تەنها ئەو تۆڕانە سکان بکە کە مۆڵەتت هەیە بۆ تاقیکردنەوەیان
- **بەکارهێنانی پیشەیی**: سەرەتا مۆڵەتی گونجاو وەربگرە

**🛡️ بە بەرپرسیارانە بەکاری بهێنە:**
- تەنها تۆڕەکانی خۆت یان ئەوانەی مۆڵەتی ڕاشکاوت هەیە سکان بکە
- ڕێزی یاساکانی خۆجێی و سیاسەتەکانی تۆڕ بگرە
- سیستەمە ئامانجەکان زۆر بار مەخەرەوە

## � دامەزراندن

### **🚀 بەڕێکردنی بێ پەیوەندی (پێشنیاری بۆ تۆڕە سنووردارەکان)**

**PowerPinger پێویستی بە دامەزراندنی نەرمەکالای زیادە نییە!** لەسەر هەر سیستەمی Windows 10+ بە بەکارهێنانی PowerShell ی ناوخۆیی کار دەکات.

**بۆ تۆڕە سنووردار/کۆمپانیاکان:**
1. **تەنها 2 فایل دابەزێنە** (~1000 هێڵ لە کۆی گشتیدا):
   - `powerPinger.ps1` (سکریپتی سەرەکی)
   - فایلی لیستی IP ەکەت (وەک `targets.csv`)

2. **ڕێگاکانی گواستنەوە**:
   - درایڤی USB
   - هاوپێچی ئیمەیڵ
   - سیستەمەکانی هاوبەشکردنی فایل
   - دەستڕاگەیشتنی مەخزەنی کۆد
   - هەر ڕێگەیەکی گواستنەوەی فایلی بەردەست

3. **یەکسەر ڕایبکە** (پێویستی بە دامەزراندن نییە):
```powershell
# ڕێگەدان بە جێبەجێکردنی سکریپت (یەک جار ڕێکخستن)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# ڕاکردنی سکەنەر - ڕێگای ١: کلیکی ڕاست
# کلیکی ڕاست لەسەر powerPinger.ps1 بکە → "Run with PowerShell" هەڵبژێرە

# ڕاکردنی سکەنەر - ڕێگای ٢: هێڵی فەرمان
.\powerPinger.ps1
```

## 📋 دۆخەکانی پشکنین

- **`ping`** - تەنها پشکنینی ping
- **`port`** - تەنها پشکنینی port  
- **`both`** - ping و port پێکەوە
- **`both`** - دۆزینەوەی تەواوی دەستڕاگەیشتنی تۆڕ

## 🛡️ دۆزینەوەی دەستڕاگەیشتنی تۆڕ

ئەم ئامرازە دەتوانێت ئەم جۆرانەی دەستڕاگەیشتن بناسێتەوە:

### پاراستنی ICMP
- **نیشانە**: خزمەتگوزارییەکان کار دەکەن بەڵام ping ناکات
- **باو لە**: تۆڕە کۆرپۆراتەکان، ژینگە سنووردارەکان

### پاراستنی خزمەتگوزاری
- **نیشانە**: ping کار دەکات بەڵام خزمەتگوزارییەکان نا
- **باو لە**: ڕێکخستنی پێشکەوتوو، چاودێری ئاسایش

### بێ سنووردارکردن
- **نیشانە**: هەردووکی ping و خزمەتگوزارییەکان کار دەکەن

## 🔧 ڕێکخستنەکان

```powershell
# کاتی چاوەڕوان
.\powerPinger.ps1 -Timeout 2000 -PortTimeout 5000

# پۆرتە تایبەتەکان
.\powerPinger.ps1 -Ports "80,443,8080,8443"

# دۆخی بازدان بەدوای شکستدا
.\powerPinger.ps1 -MaxFailures 3 -Jump 5

# سنووردارکردنی وەڵامەکان لە هەر مەودایەکدا
.\powerPinger.ps1 -MaxResponses 5
```

## 🔍 جۆرەکانی دۆزینەوەی دەستڕاگەیشتن

| دۆخ | وەسف |
|-----|------|
| **No** | هەردووکی ping و پۆرتەکان کار دەکەن |
| **ICMP-Blocked** | خزمەتگوزارییەکان کار دەکەن، ping نا |
| **Service-Blocked** | ping کار دەکات، خزمەتگوزارییەکان ڕەتدەکرێنەوە |
| **Unassigned-Range** | هیچ وەڵامێک نییە، ئەگەری سێرڤەری تەرخاننەکراو |

## 🎯 سیناریۆکانی بەکارهێنان و خۆکارکردن

### **📧 بەڕێکردنی ئیمەیڵ/USB (کەمتر لە 18KB)**
```powershell
# فایلە پێویستەکان: powerPinger.ps1 + ip_list.csv
# زیپ کراو: کەمتر لە 18KB - دەتوانرێت ئیمەیڵ بکرێت!
# دەریهێنان و ڕاکردنی یەکسەر لەسەر هەر ماشینێکی Windows
```

### **⏰ خۆکارکردنی Task Scheduler**
```powershell
# دروستکردنی ئەرکی کاتخستن بۆ پشکنینی ڕێکخستن
$action = New-ScheduledTaskAction -Execute 'PowerShell.exe' -Argument '-File "C:\path\to\powerPinger.ps1"'
$trigger = New-ScheduledTaskTrigger -Daily -At 9am
Register-ScheduledTask -Action $action -Trigger $trigger -TaskName "NetworkScan"
```

### **🔄 پشکنینی خۆکار لەگەڵ وچان**
```powershell
# پشکنین هەر کاتژمێر لەگەڵ دەرچوونی نیشاندار لەگەڵ کات
while ($true) {
    $timestamp = Get-Date -Format "yyyy-MM-dd_HH-mm"
    .\powerPinger.ps1 -OutputFile "scan_$timestamp.csv"
    Start-Sleep -Seconds 3600  # وچانی 1 کاتژمێر
}
```

### **🎮 بەکارهێنانی کارلێککاری**
```powershell
# پشکنینی خێرا لەگەڵ ڕێکخستنەکانی پێشوەخت
.\powerPinger.ps1

# پشکنینی پێشکەوتوو لەگەڵ ڕێکخستنەکانی تایبەت
.\powerPinger.ps1 -ScanMode "both" -Ports "80,443,22,53" -MaxResponses 10
```
