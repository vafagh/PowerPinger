ئامرازێکی سووکی پاوەرشێل بۆ پشکنینی تۆڕ و دۆزینەوەی نموونەکانی فیلتەرکردنی تۆڕ لە ڕێگەی ping و port scanning. **بە شێوەیەکی ڕاستەوخۆ لەسەر Windows 10+ کار دەکات بەبێ پێویستی بە دامەزراندنی نەرمەکالای زیادە** - تەواو گونجاوە بۆ بەکارهێنان لە تۆڕە سنووردارەکاندا.

## 🌟 تایبەتمەندییە سەرەکییەکان

- 🛡️ **دۆزینەوەی فیلتەرکردنی تۆڕ**: شیکردنەوەی پێشکەوتوو بۆ دۆزینەوەی جۆرە جیاوازەکانی سنووردارکردن و میکانیزمەکانی فیلتەرکردن
- 🚀 **بەبێ پێوەندی**: هیچ نەرمەکالایەکی زیادە پێویست ناکات - تەنها PowerShell
- ⚡ **خێرا و کارا**: ~1000 هێڵی کۆد + لیستی IP = کۆمەڵە تەواوی شیکردنەوەی تۆڕ
- 🎯 **چەندین دۆخی پشکنین**: ping/port/both/smart
- 📊 **راپۆرتی تەفصیلی**: دەرئەنجامەکان لە فۆرماتی CSV دا

## 🚀 دەستپێکردنی خێرا

### پێشمەرجەکان
- Windows 10 یان بەرزتر
- PowerShell 5.1+ (بە شێوەیەکی سروشتی دامەزراوە)

### 1. داگرتن
```bash
git clone https://github.com/vafagh/PowerPinger.git
cd PowerPinger
```

### 2. پشکنینی یەکەمین
```powershell
# پشکنینی سادە
.\powerPinger.ps1

# پشکنینی زیرەک
.\powerPinger.ps1 -ScanMode "smart" -Ports "80,443,22,53"
```

## 📋 دۆخەکانی پشکنین

- **`ping`** - تەنها پشکنینی ping
- **`port`** - تەنها پشکنینی port  
- **`both`** - ping و port پێکەوە
- **`smart`** - دۆزینەوەی زیرەکانەی فیلتەرکردنی تۆڕ

## 🛡️ دۆزینەوەی فیلتەرکردنی تۆڕ

ئەم ئامرازە دەتوانێت ئەم جۆرانەی فیلتەرکردن بناسێتەوە:

### پاراستنی ICMP
- **نیشانە**: خزمەتگوزارییەکان کار دەکەن بەڵام ping ناکات
- **باو لە**: تۆڕە کۆرپۆراتەکان، ژینگە سنووردارەکان

### پاراستنی خزمەتگوزاری
- **نیشانە**: ping کار دەکات بەڵام خزمەتگوزارییەکان نا
- **باو لە**: فیلتەرکردنی پێشکەوتوو، چاودێری ئاسایش

### بێ فیلتەرکردن
- **نیشانە**: هەردووکی ping و خزمەتگوزارییەکان کار دەکەن

## 🔧 ڕێکخستنەکان

```powershell
# کاتی چاوەڕوان
.\powerPinger.ps1 -Timeout 2000 -PortTimeout 5000

# پۆرتە تایبەتەکان
.\powerPinger.ps1 -Ports "80,443,8080,8443"

# دۆخی بازدان بەدوای شکستدا
.\powerPinger.ps1 -MaxFailures 3 -Jump 5

# سنووردارکردنی وەڵامەکان لە هەر مەودایەکدا
.\powerPinger.ps1 -MaxResponses 5
```

## 🔍 جۆرەکانی دۆزینەوەی فیلتەرکردن

| دۆخ | وەسف |
|-----|------|
| **No** | هەردووکی ping و پۆرتەکان کار دەکەن |
| **ICMP-Blocked** | خزمەتگوزارییەکان کار دەکەن، ping نا |
| **Service-Blocked** | ping کار دەکات، خزمەتگوزارییەکان ڕەتدەکرێنەوە |
| **Likely-Filtered** | نموونەی تێکەڵ، پێویستی بە وردبینی زیاترە |
| **Unassigned-Range** | هیچ وەڵامێک نییە، ئەگەری IPی تەرخاننەکراو |

## 🎯 چی، کەی، و بۆ کێ

### 🤔 **PowerPinger چی دەکات؟**
PowerPinger **شیکەرەوەی دەستڕاگەیشتنی تۆڕە** کە ناونیشانە تایبەتەکانی IPv4 و مەودای IPـەکان پشکنین دەکات بۆ دیاریکردنی ئەو سێرڤەر و خزمەتگوزارییانەی کە لە هەلوێستی ئێستای تۆڕەکەتەوە دەگەیشترێن. ئەو لیستێکی تەواو لە IPـی دەستڕاگەیشتوو دەربڕی دەکات لەگەڵ زانیاری تەفصیلی پەیوەندیکردن.

**ئەرکە سەرەکییەکان:**
- 🔍 **دۆزینەوەی تۆڕ**: نیسناندەری ڕاژەی زیندوو و خزمەتگوزارییە دەستڕاگەیشتووەکان
- 📊 **شیکردنەوەی پەیوەندی**: نەخشەسازی ئەو IPـانەی وەڵام دەدەنەوە بۆ ping لە بەرانبەر پەیوەندییەکانی port
- 🛡️ **دۆزینەوەی فیلتەرکردن**: ئاشکراکردنی سنووردارکردنەکانی تۆڕ و نموونەکانی دەستڕاگەیشتن
- 📋 **نەخشەسازی دەستڕاگەیشتن**: دروستکردنی راپۆرتی تەفصیلی ژێرخانی دەستڕاگەیشتوو

### ⏰ **کەی PowerPinger سوودمەندە؟**
ئەم ئامرازە بە تایبەتی بەنرخە لە **ژینگەی تۆڕی سنووردار**دا کە دەستڕاگەیشتنی ئینتەرنێتی ئاسایی سنوورداری هەیە یان فیلتەر کراوە:

- 🚫 **هیچ دەستڕاگەیشتنێکی ڕاستەوخۆی ئینتەرنێت نییە**: کاتێک تۆڕەکەت زۆربەی پەیوەندییەکانی دەرەوە بلۆک دەکات
- 🔒 **سنووردارکردنی VPN/Proxy**: کاتێک پەیوەندی VPN، proxy، یان bridge بەردەست نییە
- 🛡️ **شیکردنەوەی دیواری ئاگر**: کاتێک پێت دەوێت تێبگەیت کە چ ڕێڕەوەکانی تۆڕ کراوەن
- 🌐 **لێکۆڵینەوەی تۆڕ**: کاتی نەخشەسازی ژێرخانی دەستڕاگەیشتوو لە تۆڕە نەناسراوەکاندا
- 🔧 **چارەسەرکردنی کێشەی پەیوەندی**: کاتی تێنیشتبوونی کێشەکانی دەستڕاگەیشتنی تۆڕ
- 📡 **دۆزینەوەی ڕێڕەوی جێگرەوە**: کاتی دۆزینەوەی ڕێڕەوەکانی تۆڕی کارکەر لە نێو سنووردارکردنەکاندا

### 👥 **کێ دەبێت PowerPinger بەکاربهێنێت؟**
PowerPinger بۆ **لێکۆڵەرانی تۆڕ و پیشەیییەکان** دیزاین کراوە کە پێویستیان بە شیکردنەوەی پەیوەندی لە ژینگەی سەختدایە:

**بەکارهێنەرانی سەرەکی:**
- 🔬 **لێکۆڵەرانی تۆڕ**: شیکردنەوەی دەستڕاگەیشتنی ئینتەرنێت و نموونەکانی فیلتەرکردن
- 🛡️ **پیشەیییەکانی ئاسایش**: تێگەیشتن لە تۆپۆلۆژیای تۆڕ و کۆنترۆڵەکانی دەستڕاگەیشتن
- 🌍 **بەرگریکارانی مافەکانی دیجیتاڵ**: بەڵگەنامەکردنی سنووردارکردنەکانی تۆڕ و سەنسۆر
- 🔧 **بەڕێوەبەرانی تۆڕ**: چارەسەرکردنی کێشەکانی پەیوەندی و دەستڕاگەیشتن
- 📊 **لێکۆڵەرانی ئەکادیمی**: لێکۆڵینەوە لە ژێرخانی ئینتەرنێت و دەستڕاگەیشتن

**حاڵەتەکانی بەکارهێنان:**
- دۆزینەوەی سێرڤەری دەستڕاگەیشتوو لە دەرەوەی سنووری تۆڕی سنووردارکراوت
- نەخشەسازی ئەو مەودا IPـانەی خزمەتگوزاری دەستڕاگەیشتوویان هەیە سەرەڕای فیلتەرکردن
- دۆزینەوەی ڕێڕەوەکانی جێگرەوەی تۆڕ کاتێک دەستڕاگەیشتنی ستاندارد بلۆک کراوە
- لێکۆڵینەوە لە نموونەکانی دەستڕاگەیشتنی ژێرخانی تۆڕ
- دروستکردنی راپۆرتی تەفصیلی پەیوەندی بۆ شیکردنەوە

- **بەکارهێنانی پیشەیی**: مۆڵەتی گونجاو وەربگرە پێش پشکنینی ئەو تۆڕانەی کە خاوەنی نیت

**🛡️ ڕێنماییەکانی بەکارهێنانی بەرپرسیارانە:**
