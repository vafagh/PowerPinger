## 🎯 چه، چه وقت، و برای چه کسی

### 🤔 **این ابزار چه کاری انجام می‌دهد؟**
PowerPinger به شما کمک می‌کند **سرورها و خدمات اینترنت قابل دسترس را پیدا کنید** زمانی که دسترسی عادی اینترنت شما محدود یا مسدود است. آدرس‌های IP را اسکن می‌کند تا کشف کند کدام‌یک هنوز قابل دسترسی هستند و چه خدماتی در دسترس است.

**چه چیزی دریافت می‌کنید:**
- 📋 **فهرست IP های قابل دسترس**: کشف سرورهایی که می‌توانید به آن‌ها متصل شوید
- 🔍 **تشخیص خدمات**: بدانید کدام پورت‌ها/خدمات روی هر IP کار می‌کنند
- 🛡️ **تحلیل شبکه**: درک اینکه کدام سرورها قابل دسترسی هستند
- 📊 **گزارش‌های تفصیلی**: فایل‌های CSV با تمام اطلاعات اتصال

### ⏰ **چه زمانی باید از این استفاده کنید؟**
عالی برای موقعیت‌هایی که دسترسی اینترنت شما شدیداً محدود است:

- 🚫 **عدم دسترسی اینترنت**: شبکه شما اکثر وب‌سایت‌ها و خدمات را مسدود می‌کند
- 🔒 **VPN/Proxy مسدود**: نمی‌توانید از ابزارهای عادی دور زدن استفاده کنید
- 🛡️ **پشت فایروال**: نیاز به یافتن اتصالاتی که هنوز کار می‌کنند
- 📡 **اتصال اضطراری**: نیاز به مسیرهای جایگزین برای خدمات اینترنت
- 🔧 **عیب‌یابی شبکه**: درک مشکلات اتصال

### 👥 **این برای چه کسانی است؟**
- 🔬 **محققان**: مطالعه محدودیت‌ها و دسترسی اینترنت
- 🌍 **مراقبان دسترسی**: مستندسازی محدودیت‌های شبکه
- 🛡️ **متخصصان شبکه**: تحلیل اتصال و محدودیت‌ها
- 🔧 **مدیران سیستم**: عیب‌یابی دسترسی شبکه
- 👨‍💻 **هرکسی در شبکه‌های محدود**: یافتن اتصالات کاری اینترنت

### 📦 **چرا این ابزار؟**
**PowerPinger بهترین اسکنر شبکه موجود نیست** - اما مزایای منحصر به فردی دارد:

- 🚀 **بدون نصب**: فقط از چیزهایی استفاده می‌کند که با Windows 10/11 می‌آید
- 📧 **فوق‌العاده قابل حمل**: ابزار + فهرست IP = کمتر از 18KB هنگام زیپ شدن (می‌توان ایمیل کرد!)
- 🔓 **زمانی که دیگران کار نمی‌کنند کار می‌کند**: نیازی به حقوق مدیر یا نرم‌افزار خاص نیست
- 🎯 **هدفمند ساخته شده**: به طور خاص برای محیط‌های شبکه محدود طراحی شده
- ⚡ **فوراً آماده**: استخراج کنید و بلافاصله روی هر کامپیوتر Windows اجرا کنید

### ⚠️ **هشدارهای مهم**

**🚨 خطرات امنیتی:**
- **شما می‌توانید نظارت شوید**: مدیران شبکه می‌توانند فعالیت اسکن شما را ببینند
- **انطباق قانونی**: فقط شبکه‌هایی را اسکن کنید که مجوز آزمایش آن‌ها را دارید
- **استفاده حرفه‌ای**: ابتدا مجوز مناسب دریافت کنید

**🛡️ استفاده مسئولانه:**
- فقط شبکه‌های خود یا با مجوز صریح را اسکن کنید
- به قوانین محلی و سیاست‌های شبکه احترام بگذارید
- سیستم‌های هدف را تحت فشار قرار ندهید

## 📦 نصب

### **🚀 استقرار بدون وابستگی (توصیه‌شده برای شبکه‌های محدود)**

**PowerPinger نیازی به نصب نرم‌افزار اضافی ندارد!** روی هر سیستم Windows 10+ با استفاده از PowerShell داخلی اجرا می‌شود.

**برای شبکه‌های محدود/شرکتی:**
1. **فقط 2 فایل دانلود کنید** (~1000 خط در مجموع):
   - `powerPinger.ps1` (اسکریپت اصلی)
   - فایل فهرست IP شما (مثل `targets.csv`)

2. **گزینه‌های روش انتقال**:
   - درایو USB
   - پیوست ایمیل
   - سیستم‌های اشتراک فایل
   - دسترسی مخزن کد
   - هر روش انتقال فایل در دسترس

3. **فوراً اجرا کنید** (نیازی به نصب نیست):
```powershell
# اجازه اجرای اسکریپت (تنظیم یک‌بار)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# اجرای اسکنر - روش ۱: کلیک راست
# روی powerPinger.ps1 کلیک راست کنید → "Run with PowerShell" را انتخاب کنید

# اجرای اسکنر - روش ۲: خط فرمان
.\powerPinger.ps1
```

### **🌐 نصب مخزن کامل**

1. **کلون مخزن:**
```bash
git clone https://github.com/vafagh/PowerPinger.git
cd PowerPinger
```

2. **اطمینان از سیاست اجرای PowerShell:**
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

## 🎯 سناریوهای استفاده و اتوماسیون

### **📧 استقرار ایمیل/USB (کمتر از 18KB)**
```powershell
# فایل‌های ضروری: powerPinger.ps1 + ip_list.csv
# زیپ شده: کمتر از 18KB - قابل ایمیل!
# استخراج و اجرای فوری روی هر ماشین Windows
```

### **⏰ اتوماسیون Task Scheduler**
```powershell
# ایجاد وظیفه زمان‌بندی شده برای اسکن منظم
$action = New-ScheduledTaskAction -Execute 'PowerShell.exe' -Argument '-File "C:\path\to\powerPinger.ps1"'
$trigger = New-ScheduledTaskTrigger -Daily -At 9am
Register-ScheduledTask -Action $action -Trigger $trigger -TaskName "NetworkScan"
```

### **🔄 اسکن خودکار با وقفه**
```powershell
# اسکن هر ساعت با خروجی برچسب‌گذاری شده با زمان
while ($true) {
    $timestamp = Get-Date -Format "yyyy-MM-dd_HH-mm"
    .\powerPinger.ps1 -OutputFile "scan_$timestamp.csv"
    Start-Sleep -Seconds 3600  # وقفه 1 ساعت
}
```

### **🎮 استفاده تعاملی**
```powershell
# اسکن سریع با تنظیمات پیش‌فرض
.\powerPinger.ps1

# اسکن پیشرفته با گزینه‌های سفارشی
.\powerPinger.ps1 -ScanMode "both" -Ports "80,443,22,53" -MaxResponses 10
```

## 🚀 شروع سریع

### پیش‌نیازها
- Windows 10 یا بالاتر

### 1. دانلود
```bash
git clone https://github.com/vafagh/PowerPinger.git
cd PowerPinger
```

### 2. اولین اسکن
```powershell
# اسکن ساده
.\powerPinger.ps1

# اسکن هوشمند
.\powerPinger.ps1 -ScanMode "both" -Ports "80,443,22,53"
```

## 📋 حالت‌های اسکن

- **`ping`** - فقط اسکن ping
- **`port`** - فقط اسکن port
- **`both`** - ping و port با هم
- **`both`** - تشخیص کامل دسترسی شبکه

## 🛡️ تشخیص دسترسی شبکه

این ابزار می‌تواند این نوع دسترسی‌ها را شناسایی کند:

### مسدودیت ICMP
- **نشانه**: سرویس‌ها کار می‌کنند اما ping نه
- **رایج در**: شبکه‌های شرکتی، محیط‌های محدود

### مسدودیت سرویس
- **نشانه**: ping کار می‌کند اما سرویس‌ها نه
- **رایج در**: تنظیمات پیشرفته، نظارت امنیتی

### بدون محدودیت
- **نشانه**: هم ping و هم سرویس‌ها کار می‌کنند

.\powerPinger.ps1 -MaxFailures 3 -Jump 5

.\powerPinger.ps1 -MaxResponses 5

## 🔍 انواع تشخیص دسترسی

| وضعیت | توضیح |
|-------|-------|
| **No** | هم ping و هم پورت‌ها کار می‌کنند |
| **ICMP-Blocked** | سرویس‌ها کار می‌کنند، ping نه |
| **Service-Blocked** | ping کار می‌کند، سرویس‌ها رد می‌شوند |
| **Unassigned-Range** | هیچ پاسخی نیست، احتمال سرور تخصیص نیافته |
